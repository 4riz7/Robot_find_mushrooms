# 🍄 Robot Find Mushrooms

Симуляция автономного робота, который перемещается по лесу, находит и собирает грибы, а затем возвращается на базу. Проект написан на C++ и демонстрирует работу с конечными автоматами, простыми алгоритмами поиска и объектно-ориентированным дизайном на реальной, пусть и игровой, задаче.

## 🎯 О проекте

Робот появляется в лесу с грибами трёх типов — **съедобными**, **галлюциногенными** и **несъедобными** — и должен:

1. найти ближайший гриб;
2. дойти до него;
3. определить его тип и решить, забирать или пропустить;
4. повторять, пока не заполнятся оба отсека (для съедобных и галлюциногенных грибов);
5. вернуться на стартовую позицию, когда отсеки полны или грибы закончились.

Всё происходит в реальном времени в консоли: на каждом шаге перерисовывается карта леса с текущей позицией робота.

## ✨ Функциональность

- Генерация леса заданного размера со случайной расстановкой грибов трёх типов
- Поиск ближайшего гриба по евклидову расстоянию и пошаговое перемещение к нему
- Определение типа гриба и распределение по отдельным отсекам с ограниченной вместимостью
- Возвращение на базу при заполнении отсеков или отсутствии грибов в лесу
- Эмуляция расхода заряда батареи с автоматическим отключением робота при разрядке
- Случайные сбои (0.1% шанс на шаг) с корректным завершением работы
- Визуализация состояния леса и робота в консоли на каждом шаге

## ⚙️ Как это устроено

Поведение робота управляется конечным автоматом:

```
TURN_ON → GOING → (поиск и сбор грибов) → GO_HOME → TURN_OFF
```

Основные сущности:

| Сущность | Назначение |
|---|---|
| `Forest` | хранит карту леса, список грибов, отвечает за их размещение, поиск ближайшего и удаление собранного |
| `Mushroom` | структура с типом и координатами гриба |
| `Robot` | хранит координаты, заряд батареи, содержимое отсеков; управляет переходами между состояниями и движением |
| `StateRobot` | перечисление состояний робота |

## 🛠 Технологии

- **Язык:** C++ (MSVC v143 / Visual Studio 2022)
- **Стандартная библиотека:** `iostream`, `vector`, `string`, `cmath`, `cstdlib`, `limits`
- **Многопоточность:** `<thread>`, `<chrono>` — для задержек между шагами симуляции

## 🚀 Запуск

**Visual Studio (рекомендуется):**
1. Открыть `Robot_find_mushrooms.vcxproj`
2. Собрать и запустить (F5)

**Через g++ (Linux/macOS):**
```bash
g++ -std=c++17 main.cpp Sleep.cpp -o robot
./robot
```
> На Linux/macOS вызов `system("cls")` для очистки консоли не сработает — см. раздел «Планы по развитию».
